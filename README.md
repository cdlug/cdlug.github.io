
# Website of Chengdu GNU/Linux User Group -- CDLUG

## 简介 

这里是 [CDLUG 的官方网站](https://cdlug.org)代码，CDLUG 是主要由成都本地的 GNU/Linux 用户结成的小组，我们有线上讨论和线下活动，讨论的内容主要是 GNU/Linux 以及技术相关的主题。更多介绍请看[关于页面](https://cdlug.org/about)
